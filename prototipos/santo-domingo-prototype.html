📝 Notas para el equipo

  • 🎨 Logo: el círculo "SDG" es un placeholder. Reemplazar con el logo real de Santo Domingo de Guzmán cuando lo consigan. Usar <img src="logo.png"> en lugar del div
  • 🎨 Paleta: toda la paleta está en variables CSS al inicio (--dom-*). Para cambiar tonos, solo editar esas variables
  • 🔤 Fuentes: Playfair Display (títulos) + Source Sans 3 (cuerpo). Se cargan desde Google Fonts. En Moodle, usar la tipografía del theme base o cargar las fuentes vía styles.css
  • 📊 Chart.js: las barras de progreso semanal son CSS puro aquí. En el plugin real, reemplazar con Chart.js bar horizontal usando los mismos colores
  • 🏆 Niveles dominicanos: los nombres de nivel (Predicador, etc.) son opcionales. Se pueden personalizar desde la config del plugin por institución
  • 📱 Responsive: ya funciona en móvil (sidebar baja al fondo). En Moodle, el bloque lateral se adapta automáticamente
